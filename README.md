# nbcamp-todo
투두리스트를 위한 API서버


## 사용 기술
- Java
- Spring Data Jpa
- Spring Security
- Mysql


## 구현 기능
- 회원가입, 로그인
- 할 일 작성, 수정, 조회, 완료
- 댓글 작성, 수정, 삭제
